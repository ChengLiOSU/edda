Detailed Description

template<typename T>
class edda::Dataset< T >

Holds all information of a dataset, which includes: 1) Geometry and 2) Data array.

You may use make_Dataset() to create a Dataset with shorter codes.

Parameters
TThe return type of at_phys() and at_comp()
